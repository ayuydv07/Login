# Login UI

A clean, responsive login interface built with Flutter. This project focuses on a simple aesthetic, utilizing standard email/password inputs and social authentication buttons.

## Features

* **Simple Authentication UI:** Clean layout using `Column`, `Padding`, and `SizedBox` for spacing.
* **Social Sign-in:** Integrated Google and Facebook buttons with monochromatic styling to match the blue theme.
* **Responsive Design:** Utilizes `double.infinity` widths to ensure buttons adapt to different mobile screen sizes.

## Technical Details

* **Framework:** Flutter
* **Language:** Dart
* **Key Dependencies:** `font_awesome_flutter` (for social icons).

   ```bash

   ```

2. **Install Dependecies:**
